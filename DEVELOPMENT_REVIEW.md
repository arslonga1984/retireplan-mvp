# 현재 개발 버전 리뷰 (기능/UX/UI 개선안)

## 1) 현재 버전 요약
- MVP는 랜딩 → 4단계 입력 → 결과 대시보드 흐름이 명확하게 구성되어 있으며, 투자 목표(수익률/MDD) 기반 전략 추천과 은퇴 후 인출 시뮬레이션까지 포함합니다.
- `zustand persist`로 입력값이 저장되어 사용자가 이탈 후에도 이어서 진행할 수 있습니다.
- 결과 화면에서 갭 분석(희망 생활비 대비 부족액), 전략/차트/What-if를 함께 제공해 의사결정 맥락이 좋은 편입니다.

## 2) 잘된 점 (유지/강화 권장)
1. **입력 흐름의 학습 비용이 낮음**
   - Step 기반 폼 + 진행바로 사용자가 현재 위치를 쉽게 이해합니다.
2. **추천 전략의 설명 가능성**
   - "AI 추천"과 추천 사유 노출, 대안 전략 비교(최대 3개) 제공은 신뢰 형성에 유리합니다.
3. **은퇴 소득 관점의 갭 분석**
   - 국민연금 포함 총소득 기준으로 부족분을 보여주는 구조가 실사용자 관점에 맞습니다.

## 3) 핵심 개선 과제 (우선순위)

### P0 (정확도/신뢰도)
1. **시나리오 엔진 고도화 (결정론 → 확률 기반 보강)**
   - 현재는 고정 수익률 ±2% 시나리오 기반이라 시장 변동성/수익률 분포를 충분히 반영하지 못합니다.
   - 개선안:
     - 월별 수익률 분포(정규/로그정규 가정) 기반 Monte Carlo 모드 추가.
     - 전략별 변동성, 상관계수(주식/채권/금) 입력값을 두고 성공확률 재계산.
     - 결과 카드에 "가정" 섹션(인플레, 기대수익률, 변동성)을 명시.

2. **은퇴 후 인출 로직 정밀화**
   - 현재 은퇴 후 월 인출은 `targetRetirementIncome`이 있으면 해당 금액으로 고정되므로, 물가연동 토글과 실제 표시값의 직관성이 떨어질 수 있습니다.
   - 개선안:
     - 실질/명목 모드를 분리해 UI에 명확히 표기.
     - 인플레 반영 시 "연도별 인출액 증가" 그래프(현재가치 vs 명목가치) 추가.

### P1 (제품 완성도)
3. **입력 검증/가이드 강화**
   - Step1 금액 입력은 자유도가 높아 오입력(0이 하나 빠짐/추가됨) 위험이 있습니다.
   - 개선안:
     - 입력 하단에 자동 단위 변환 표시 (예: `12,000,000원 (약 1,200만원)`).
     - 비정상 값 감지 시 경고 배지(예: 월 납입액이 월 소득 대비 과도한 경우).

4. **전략 선택 화면의 스캔성 개선**
   - 전략 카드 정보량이 많아 모바일에서 피로도가 높습니다.
   - 개선안:
     - 카드 기본은 "핵심 3지표"만 노출, 상세는 아코디언으로 펼침.
     - 비교 체크박스를 카드 하단 고정 액션으로 이동해 오조작 감소.

5. **결과 화면의 실행 액션 강화**
   - 현재는 분석 정보 제공은 좋지만 "다음 행동" 유도가 상대적으로 약합니다.
   - 개선안:
     - "부족분 해소 시뮬레이션" CTA(월 납입액 +10만/+20만, 은퇴시기 +2년 등) 프리셋 버튼.
     - 선택한 시나리오를 PDF/이미지로 저장 또는 공유 링크 생성.

### P2 (확장성/운영)
6. **개인화 루프 추가**
   - 사용자 세그먼트(초보/중급/고위험 선호)에 맞춘 설명 톤/추천 깊이 차등 제공.

7. **신뢰 신호 강화**
   - 전략 성과 근거 데이터 출처, 백테스트 기간/한계, "투자 권유 아님" 고지 영역을 상시 노출.

## 4) UI 개선안 (컴포넌트 단위)
1. **랜딩 페이지**
   - 개선: Hero CTA 하단에 샘플 결과 스냅샷(미리보기 카드) 추가.
   - 기대효과: 첫 방문 전환율 개선.

2. **Step 1 (기본정보)**
   - 개선: 숫자 입력 포커스 시 숫자 키패드 최적화 + 최근 입력값 undo 버튼.
   - 개선: 국민연금 입력에 "국민연금공단 조회 링크" 보조 텍스트 추가.

3. **Step 2 (목표설정)**
   - 개선: 수익률/MDD 조합별 "위험 성향 라벨"(안정/중립/공격) 즉시 표시.

4. **Step 3 (전략추천)**
   - 개선: 추천/대안/전체 탭으로 구획, 모바일에서 긴 스크롤 압축.
   - 개선: 비교 선택 시 하단 sticky bar에 "비교하기(2/3)" 노출.

5. **Step 4 (연금수령)**
   - 개선: 종신형 vs 확정기간형에 대해 "장단점 카드"를 먼저 제시하고 선택 유도.

6. **대시보드**
   - 개선: 요약 카드 상단에 "한 줄 결론" 배지(예: "목표 대비 월 45만원 부족").
   - 개선: 탭 구조에 "리스크" 탭 추가(MDD, 변동성, 고갈 가능 시점).

## 5) 빠른 실행안 (2주)
- Week 1
  - 입력 단위 가이드/경고 배지 추가.
  - 전략 카드 정보 밀도 축소 + 상세 펼침 구조로 변경.
- Week 2
  - 결과 화면 CTA(부족분 해소 프리셋) 추가.
  - 결과 공유(이미지 저장) 1차 버전 추가.

## 6) 제안 KPI
- 퍼널 전환율: 랜딩 → Step1 시작, Step4 → 결과 도달률
- 입력 이탈률: Step1/Step3 구간 이탈 감소율
- 리포트 활용도: 결과 공유/저장 클릭률
- 행동 유도 성과: "부족분 해소 프리셋" 실행 비율

## 7) 결론
- 현재 버전은 "빠른 입력 + 즉시 인사이트"라는 MVP 목적을 이미 잘 달성했습니다.
- 다음 단계는 **정확도 고도화(P0)**와 **행동 유도 UX(P1)**를 동시에 추진하는 것이 가장 큰 임팩트를 냅니다.
